# GitHub Readme Stats

📊 Generate beautiful stats cards for your GitHub profile README

## Features

- Total stars earned
- Total commits
- Contribution streak
- Top languages
- Profile views counter

## Usage

Add this to your README.md:

``markdown
![Stats](https://github-readme-stats.vercel.app/api?username=akashhg2007)
`` 

## Installation

1. Fork this repository
2. Deploy to Vercel
3. Use the URL in your README
